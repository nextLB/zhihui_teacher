关于系统

系统介绍

本系统是一个面向智慧课堂的教师语音特征可视化与风格画像系统。通过融合语音特征分析与教师姿态行为识别，生成综合性的教师风格画像。

主要功能

  • 语音分析：支持音频文件上传和分析，提取语速、语气、提问类型等特征
  • 行为分析：采用YOLOv8进行视频分析，识别教师姿态和学生行为
  • 风格画像：融合多维度特征，生成教师风格画像和可视化图表

技术栈

  • 后端：Python + Django
  • 语音处理：librosa, Vosk/Whisper
  • 图像处理：OpenCV + YOLOv8
  • 前端：HTML + CSS + JavaScript + ECharts
